# Data

## Writing_Assessment_Cleaned_Dataset_Tool_Specify.csv

Pre-extracted linguistic features for 85 college student writing samples.

### Variables

- `P#`: Participant ID
- `Primary_Language`: 0 = Native English, 1 = Non-native English
- `Writing_Quality_Sum_Score`: Human-rated quality (WQS-HE), range 10.6â€“21.6
- `OST_*`: Open-source toolkit features (30 variables)
- `TERA_*`: T.E.R.A. features (7 variables)
- `LCA_*`: Lexical Complexity Analyzer features (33 variables)
- `SCA_*`: Syntactic Complexity Analyzer features (23 variables)

### Writing Task

Prompt: "Describe a valuable experience you have had during your college career."  
Time limit: 5 minutes  
Assessment: Writing Quality Scale for Higher Education (WQS-HE; Stuart & Barnett, 2024)
